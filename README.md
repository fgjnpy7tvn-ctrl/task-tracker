# Task Tracker (CLI)

A simple command-line task tracker written in Python.
Stores tasks locally in a text file.

## Features
- View tasks
- Add tasks
- Mark tasks as done
- Delete tasks
- Save & load from file

## How to Run

1. Open a terminal in the project folder
2. Run the program with:

py task_tracker.py

## Notes
- Tasks are saved locally in a text file
- The file is ignored by git using .gitignore
